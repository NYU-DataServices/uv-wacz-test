# uv-wacz-test

Crawl live sites to produce WACZ files with local [browsertrix-crawler](https://crawler.docs.browsertrix.com/) via Docker. 

Results are saved to `/crawls/collections` for demo/test use in UltraViolet as part of WebRecorder playback custom previewer development (WiP, 2025; results .gitignored due to size)
